# Smart To-Do App

A modern To-Do application with clean UI and useful features.

## 🚀 Features
- Add, delete, complete tasks
- Priority levels (High, Medium, Low)
- Dark mode toggle 🌙
- Local storage (data saved after refresh)
- Smooth animations & task cards UI

## 🛠 Tech Used
- HTML
- CSS
- JavaScript
